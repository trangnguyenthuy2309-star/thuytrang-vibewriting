## 📋 WORKFLOW TUẦN (7 bước bắt buộc)

### **BƯỚC 1: ĐỌC STRATEGY (Bắt buộc lần đầu, review hàng tháng)**

Trước khi bắt đầu sản xuất tuần đầu tiên hoặc khi thay đổi chiến lược:

- [ ] Mở `1_FOUNDATION/01_STRATEGY_OVERVIEW.md`
- [ ] Đọc: 4 Cột trụ, 5 Tuyến content, Timeline, KPI targets
- [ ] Hiểu: Audience pain points, goals, brand voice
- [ ] Ghi chú: Những insights quan trọng
- [ ] Xác nhận: "Tôi hiểu rõ strategy của dự án"

**Thời gian:** 30 phút (lần đầu), 15 phút (review)

---

### **BƯỚC 2: BRAINSTORM & UPDATE IDEAS BANK (Thứ 2, 60 phút)**

#### **2.1: Đọc Ideation System**

- [ ] Mở `3_IDEATION/3.1_IDEATION_SYSTEM.md`
- [ ] Hiểu 5 phương pháp tìm ý tưởng:
  - [ ] Daily Observation (quan sát hàng ngày)
  - [ ] Audience Questions (câu hỏi từ audience)
  - [ ] Content Swipe (hooks từ creators khác)
  - [ ] Pillar-based Brainstorm (ideate theo pillar)
  - [ ] Trend + Seasonal (xu hướng, sự kiện)

#### **2.2: Ghi ý tưởng mới**

Làm ngay khi có ý tưởng trong tuần (hoặc tổng hợp vào sáng thứ 2):

- [ ] Mở `3_IDEATION/3.2_CONTENT_IDEAS_BANK.csv`
- [ ] Ghi các cột (bắt buộc):
  - [ ] Ngày (hôm nay)
  - [ ] Chủ đề (1-2 từ ngắn gọn)
  - [ ] Cột trụ (1, 2, 3, hoặc 4)
  - [ ] Tuyến (1, 2, 3, 4, hoặc 5)
  - [ ] Nguồn (Quan sát, Audience, Swipe, Sách, Xu hướng)
  - [ ] Hook (dòng mở bài)
  - [ ] Triển khai (tóm tắt body)
  - [ ] Status: "Idea Bank" (mới thêm)
  - [ ] Ghi chú (thêm chi tiết)
- [ ] Lưu file

#### **2.3: Chọn 7 ý tưởng tốt nhất**

- [ ] Lọc ideas trong tuần (hoặc xem tất cả từ Ideas Bank)
- [ ] Chọn 7 ideas tốt nhất (diverse pillars + tuyến)
- [ ] **Cập nhật Status:** Từ "Idea Bank" → "Brainstorm ✓"
- [ ] Lưu file

**Kết quả:** 7 ideas ready cho tuần viết (Tuyến 1-3: 4-5 bài, Tuyến 4-5: 2-3 bài)

---

### **BƯỚC 3: LỰA CHỌN & ĐỌC CONTENT CALENDAR (Thứ 2, 30 phút)**

- [ ] Mở `4_PRODUCTION/4.4_TEMPLATE_CONTENT_CALENDAR_4_TUAN.md`
- [ ] Copy 7 ý tưởng từ Ideas Bank vào calendar tuần này
- [ ] Phân bố từng bài theo ngày:
  - [ ] Facebook long-form: Thứ 2, 3, 4 (hoặc tuỳ)
  - [ ] TikTok short-form: Thứ 3, 5, 6 (hoặc tuỳ)
  - [ ] Mix: Theo ratio 3-4 Facebook + 3-4 TikTok/tuần
- [ ] Xác nhận: Content mix đúng (Story 50%, Tips 30%, Mindset 20%)
- [ ] Lưu file

**Kết quả:** Calendar rõ ràng, biết viết cái gì mỗi ngày

---

### **BƯỚC 4: ĐỌC PRODUCTION GUIDES (Bắt buộc trước khi draft - 90 phút)**

Đọc tất cả 4 file này TRƯỚC khi mở máy viết draft:

#### **4.1: Đọc Framework 5 Tuyến**

- [ ] Mở `4.1_FRAMEWORK_VIET_BAI_5_TUYEN.md`
- [ ] Đọc structure cho mỗi tuyến:
  - [ ] Tuyến 1 (Stories 35%): Hook → Setup → Conflict → Realization → Lesson → Invitation
  - [ ] Tuyến 2 (Mindset 15%): Old Belief → Shift → Evidence → New Practice → Lesson
  - [ ] Tuyến 3 (Tips 20%): Problem → 3 Solutions → Why → CTA
  - [ ] Tuyến 4 (Balance 15%): Story + Insights + Invitation
  - [ ] Tuyến 5 (Philosophy 15%): Deep thinking + Reflection + Questions
- [ ] Hiểu templates & ví dụ cho từng tuyến
- [ ] Note: Platform differences (Facebook long vs TikTok short)

#### **4.2: Đọc Quy Trình Sản Xuất Hàng Tuần**

- [ ] Mở `4.2_QUY_TRINH_SANXUAT_HANG_TUAN.md`
- [ ] Hiểu 6 bước:
  - [ ] Step 1: Pre-writing (warm up, review framework)
  - [ ] Step 2: Draft (viết nhanh, không edit)
  - [ ] Step 3: Edit (refine từng paragraph)
  - [ ] Step 4: Brand voice check
  - [ ] Step 5: Format (tuỳ platform)
  - [ ] Step 6: Final review
- [ ] Biết: Thời gian mỗi bài (~45-60 phút)

#### **4.3: Đọc Brand Voice Guide**

- [ ] Mở `4.3_BRAND_VOICE_GUIDE.md`
- [ ] Review 5 attributes:
  - [ ] Chân thành (authentic)
  - [ ] Sâu sắc (deep)
  - [ ] Dễ áp dụng (practical)
  - [ ] Me-to-You (conversational)
  - [ ] Evidence-based (có nguồn)
- [ ] Nhớ "AVOID" list: Formal, khoa học, khoe bản thân, phán xét, ẩu, drama
- [ ] Print/bookmark: Tone checklist (sẽ dùng khi edit)

#### **4.4: Đọc Template Content Calendar**

- [ ] Mở `4.4_TEMPLATE_CONTENT_CALENDAR_4_TUAN.md`
- [ ] Hiểu cấu trúc calendar
- [ ] Check mix percentages (đã làm ở Bước 3)

**Ghi chú:** Bước 4 này là "warm-up" trí não. Bạn sẽ viết tốt hơn khi đã review framework vào đầu tuần.

---

### **BƯỚC 5: VIẾT DRAFTS (Thứ 3-6, 5-7 tiếng)**

Với mỗi bài viết:

#### **5.1: Tạo sub-folder cho draft**

- [ ] Mở folder `4_PRODUCTION/4.5_WEEKLY_DRAFTS/`
- [ ] Tạo **NEW FOLDER** với cấu trúc:
  ```
  [NGÀY]_[HOOK_CHÍNH]
  
  Ví dụ:
  2024-01-15_Con_hoi_cau_noi_dau
  2024-01-16_Toi_tung_tin_phai_la_me_hoan_hao
  2024-01-17_3_cach_quan_ly_thoi_gian
  ```

#### **5.2: Viết draft**

- [ ] Mở framework tương ứng từ `4.1_FRAMEWORK_VIET_BAI_5_TUYEN.md`
- [ ] Viết nhanh (không edit trong lúc viết)
- [ ] Copy template → Fill in sections
- [ ] Lưu file: `DRAFT_[NGAY].md` hoặc `DRAFT_[NGAY].txt`
  - Ví dụ: `DRAFT_2024-01-15.md`

#### **5.3: Update status ý tưởng**

- [ ] Mở `3.2_CONTENT_IDEAS_BANK.csv`
- [ ] Tìm ý tưởng vừa draft
- [ ] **Cập nhật Status:** Từ "Brainstorm ✓" → "Draft ✓"
- [ ] Lưu file

**Kết quả:** 7 drafts trong 7 sub-folders, tất cả status updated

---

### **BƯỚC 6: EDIT & QC (Thứ 5-6 chiều, 2-3 tiếng)**

Với mỗi draft:

#### **6.1: Self-edit**

Mở draft và làm từng bước:

- [ ] **Đọc lại toàn bộ** (để catch typo, grammar)
- [ ] **Check structure:** Có đủ sections theo framework không?
- [ ] **Check hook:** Hook có attractive không? (Dừng scroll được không?)
- [ ] **Check evidence:** Có link, sách, hoặc source không?
- [ ] **Check CTA:** Có invitation/question ở cuối không?
- [ ] **Check length:** Phù hợp platform không? (FB long, TikTok short)

#### **6.2: Brand voice check**

Dùng checklist từ `4.3_BRAND_VOICE_GUIDE.md`:

- [ ] Conversational (không formal)?
- [ ] Specific (không chung chung)?
- [ ] Vulnerable (không khoe)?
- [ ] Has evidence (không đoán)?
- [ ] Me-to-you (không expert-to-student)?
- [ ] Không dùng từ AVOID

#### **6.3: Platform format**

- [ ] **Facebook:** Paragraph rõ ràng, dòng mới tạo, line breaks hợp lý
- [ ] **TikTok:** Ngắn gọn, dễ đọc, visual friendly (emoji, dấu, symbols)
- [ ] **Cả 2:** Hook mạnh, CTA rõ

#### **6.4: Final touches**

- [ ] Add emoji (nếu cần, nhất là TikTok)
- [ ] Check: Không có typo, không có dấu lạ
- [ ] Add hashtag (nếu dùng): Không quá nhiều, relevant
- [ ] Save: Finalized version (rename: `FINAL_[NGAY].md`)

**Kết quả:** 7 drafts edited, ready to publish

- [ ] Note: Post time (khi nào post)

- [ ] Mở `3.2_CONTENT_IDEAS_BANK.csv`
- [ ] **Cập nhật Status:** Từ "Draft ✓" → "Posted ✓" + ngày post
- [ ] Lưu file

#### **7.3: Weekly tracking**

- [ ] Mở `5_ANALYTICS/5.1_KPI_TRACKING_SHEET.md`
- [ ] Ghi:
  - [ ] Ngày post (7 bài)
  - [ ] Platform (FB hoặc TikTok)
  - [ ] Hook/Title
  - [ ] URL (nếu public)
- [ ] Lưu file

#### **7.4: Monitor (Hàng ngày)**

- [ ] Check comments (sáng & chiều)
- [ ] Reply audience (conversational)
- [ ] Note: Comments quan trọng để refine ideas

## 🚨 RED FLAGS - STOP & REVIEW NẾU:

- [ ] Chưa đọc Strategy → STOP, đọc ngay
- [ ] Chưa update Ideas Bank status → STOP, update trước viết
- [ ] Chưa đọc Framework → STOP, draft sẽ tệ hơn
- [ ] Draft không có hook mạnh → Revise hook, không publish
- [ ] Draft quá formal hoặc khoa học → Edit lại cho conversational
- [ ] Không có CTA hoặc invitation → Add ngay trước publish
- [ ] Ideas Bank status vẫn "Brainstorm" khi publish → Update!

## 💡 TIPS THỰC HÀNH

1. **In checklist này out & dán trên bàn** - Nhìn sẽ nhớ đừng bỏ bước
2. **Bookmark 4 files production** - Dễ mở nhanh
3. **Set calendar reminders:** Thứ 2 (brainstorm), Thứ 3 (start draft), Thứ 5 (edit), Thứ 7 (publish)
4. **Voice note ideas** - Ghi voice vào điện thoại, sau vào Ideas Bank
5. **Track trong 1 tháng** - Thấy pattern nào hay, làm nhiều hơn
